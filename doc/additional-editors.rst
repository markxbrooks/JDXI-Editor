Additional Editors
==================

Advanced editing tools and specialized interfaces for professional sound design and performance.

Available Editors
=================

* :doc:`Arpeggiator Editor <arpeggiator>` — Sophisticated pattern generation and sequencing
* :doc:`Effects Editor <effects>` — Master effects processing with reverb, delay, compressor, phaser
* :doc:`Vocal Effects Editor <vocal_effects>` — Vocoder and Auto-Pitch (OFF/VOCODER/AUTO-PITCH)
* :doc:`Pattern Sequencer <pattern_sequencer>` — 4×16 step grid, MIDI load/save
* :doc:`MIDI File Player <midi_file_player>` — Track classification, channel assignment, playback

**Program Editor Tabs**
   The Program Editor includes a **System Settings** tab for Master Tune, Master Level, Key Shift, RX/TX Program Change/Bank Select, Keyboard Velocity, and Velocity Curve.
