# Manual Tk Installation Instructions for macOS

## Option 1: Use CPAN with proper configuration

1. Set environment variables:
   export TCL_PREFIX=$(brew --prefix tcl-tk)
   export TK_PREFIX=$(brew --prefix tcl-tk)

2. Start CPAN:
   cpan

3. In CPAN shell, configure and install:
   o conf makepl_arg "TCL_PREFIX=$TCL_PREFIX TK_PREFIX=$TK_PREFIX"
   o conf commit
   install Tk

## Option 2: Download and install manually

1. Download Tk from CPAN:
   cd ~/Downloads
   cpan -g Tk

2. Extract and configure:
   tar -xzf Tk-*.tar.gz
   cd Tk-*
   perl Makefile.PL TCL_PREFIX=$(brew --prefix tcl-tk) TK_PREFIX=$(brew --prefix tcl-tk)

3. Build and install:
   make
   make test
   make install

## Option 3: Use system Perl (if available)

Some macOS systems have Tk pre-installed. Try:
   /usr/bin/perl -MTk -e 'print "Tk available\n"'

If that works, you can use system Perl instead of Homebrew Perl.
